# Shape Interact — Controls & Input Guide

A quick reference for how to interact with the scene using touch gestures and the on-screen buttons.

---

## Default State (on launch)

When the game starts, only these controls are active:

- **3-finger swipe left / right** — rotates the camera around the scene
- **Tap an object** — selects it and enables object-level controls

> ⚠️ Camera movement, camera rotation, zoom, and gyroscope are all **OFF** at launch. Use the buttons below to enable them.

---

## Interacting With Objects

| Gesture | Action |
|---|---|
| Tap object | Select it (colour changes to confirm) |
| Tap empty space | Deselect current object |
| 1-finger drag *(selected)* | Move object across the surface |
| 2-finger pinch *(selected)* | Scale object smaller / larger |
| 2-finger twist *(selected)* | Rotate object |

---

## On-Screen Buttons

Each button toggles a camera feature on or off. The button label updates to show the current state.

### Camera Movement
- **Toggles:** Camera Movement ON / OFF
- **When ON:** drag with 2 fingers to pan the camera left, right, up, or down

### Camera Rotation
- **Toggles:** Camera Rotation ON / OFF
- **When ON:** 2-finger twist rotates the camera around its own axis

### Camera Zoom
- **Toggles:** Camera Zoom ON / OFF
- **When ON:** 2-finger pinch in / out zooms the camera

### Gyroscope
- **Toggles:** Gyro ON / OFF
- **When ON:** rotate your phone left and right to rotate the camera. Any object should be selected first to activate the rotation.

---

## Resetting the Scene

If any button or gesture stops responding, tap the **Reset** button to reload the scene.

- All object positions and scales are restored
- All buttons reset to OFF (3-finger camera rotation stays active)
- Any selected object is deselected

> 💡 If a button does not respond on first tap, reset the scene and try again.

---

## Quick Reference

| Gesture | Result | Requires |
|---|---|---|
| 1-finger tap | Select / deselect object | Always on |
| 1-finger drag | Move selected object | Always on |
| 2-finger pinch | Scale object | Object selected |
| 2-finger twist | Rotate object | Object selected |
| 3-finger swipe L/R | Rotate camera | Always on |
| 2-finger drag | Pan camera | Camera Movement ON |
| 2-finger twist | Rotate camera | Camera Rotation ON |
| 2-finger pinch | Zoom camera | Camera Zoom ON |
| Tilt phone L/R | Gyroscope camera rotate | Gyro ON |
